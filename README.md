"# Invariant-Finder-for-Energy"
A program code for finding invariants or checking a combination for invariance. Allows expansion of the thermodynamic potential only to the first power.
Mode: 0 - basic mode, 1 - single combination check mode.
The user needs to enter the Turov cipher (which corresponds to the magnetic structure), and also specify a combination of vector products (which determines the physical effect).
Important: If you want to work in mode 1, you must also change vectors_input to the appropriate vectors. The terms are always separated by "+"; the minus sign appears before the first component of a negative term.
